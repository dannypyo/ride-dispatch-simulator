# Ride Dispatch Simulator (with Generators)

A simulation of a ride-sharing dispatch system (like Uber or Lyft) built using Python generators. This project demonstrates how generators can model real-world systems by pausing and resuming execution.

## Description
This program simulates how a ride dispatch system works:

- Riders continuously request rides
- Drivers become available and are assigned rides
- The system matches riders with drivers in real time

Using generators (yield), the simulation mimics an event-driven system where different processes (ride requests and driver availability) happen over time.
